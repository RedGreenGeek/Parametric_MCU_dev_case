# Parametric_MCU_dev_case
This repo contains a parametric freecad model for MCU development boards.

Currently the following boards are supported.
* ESP32 NodeMCU Dev Kit C v2
* Xiao nRF52840
* Xiao RP2040
* NodeMCU ESP32-WROOM-32D
* Xiao RP2350

Documentation is in the [Labitat wiki](https://labitat.dk/wiki/Parametric_MCU_dev_case)

![Render for large module](https://github.com/RedGreenGeek/Parametric_MCU_dev_case/blob/main/ESP32_NodeMCU_DevKit_C_v2_Case.png?raw=true)

# How to
Change the board, and features in "VarSet".
